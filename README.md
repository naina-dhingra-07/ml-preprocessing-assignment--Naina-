# Machine Leanring Preprocessing Assignment
The dataset (called insurance.csv) has been picked from kaggle. It is a vast dataset consisting of 1300+ rows and 7 columns.
It provides information regarding gender, age, body mass index of people from various regions (such as north-west, south-east etc). It also includes whether the person smokes and how many children they have. All these columns are further used to predict insurance charges.
For now we have only worked on cleaning the data to prepare it to train machine learning models.
